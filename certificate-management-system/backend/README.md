# Backend Structure

## Folders
- `config/`: runtime configuration (`db.js`)
- `controllers/`: route handlers
- `middleware/`: auth, upload, error middleware
- `models/`: mongoose models
- `routes/`: API route modules
- `scripts/`: admin/data seed and repair scripts
- `utils/`: helper utilities
- `uploads/`: uploaded files

## Script Commands
- `npm run dev`
- `npm run seed:admin`
- `npm run seed:admin:master`
- `npm run repair:admin`
- `npm run seed:certificates`
- `npm run seed:demo`
